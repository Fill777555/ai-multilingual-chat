Многоязычный чат с автопереводом через AI для поддержки клиентов на разных языках.

== Description ==

AI Multilingual Chat - это мощный плагин для WordPress, который обеспечивает многоязычную поддержку клиентов через интегрированный чат с автоматическим переводом сообщений.

**Основные возможности:**

* Автоматический перевод сообщений между клиентом и администратором
* Поддержка множества языков (русский, английский, украинский, испанский, немецкий, французский и др.)
* Звуковые уведомления для администратора и клиентов
* Выбор темы (светлая/темная)
* Выбор эмодзи в чате
* Экспорт переписки в CSV
* Настраиваемый виджет чата
* Адаптивный дизайн

== Installation ==

1. Загрузите папку плагина в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Настройте плагин в разделе 'AI Chat' в админ-панели
4. Добавьте API ключ для перевода
5. Настройте параметры виджета чата

== Frequently Asked Questions ==

= Как добавить API ключ для перевода? =

Перейдите в раздел 'AI Chat' -> 'Настройки' и добавьте ваш API ключ в соответствующее поле.

= Поддерживает ли плагин темную тему? =

Да, вы можете включить темную тему в настройках плагина.

= Как клиенты могут отключить звуковые уведомления? =

Клиенты могут нажать на кнопку динамика в заголовке чата для включения/отключения звуковых уведомлений.

== Technical Details ==

Плагин использует WordPress AJAX API для обмена сообщениями в реальном времени, Web Audio API для звуковых уведомлений, и интеграцию с AI сервисами для автоматического перевода.
